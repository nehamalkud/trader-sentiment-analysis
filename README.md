# 📊 Trader Performance vs Market Sentiment Analysis

## 📌 Project Overview
This project analyzes how market sentiment (Fear vs Greed) impacts trader performance using real trading data.

---

## 📂 Dataset
- Historical trading data (PnL, trades, timestamps)
- Market sentiment data (Fear & Greed index)

---

## ⚙️ Workflow
1. Data Cleaning  
2. Feature Engineering  
3. Merging datasets based on date  
4. Exploratory Data Analysis  
5. Visualization  

---

## 📊 Key Insights
- 📈 Traders earn higher profits during **Greed** sentiment  
- 📉 **Fear** periods show lower win rates  
- 🔁 Trading activity increases during volatile conditions  

---

## 📌 Strategy Recommendations
- Reduce exposure during Fear periods  
- Apply trend-following strategies during Greed  
- Avoid overtrading in uncertain markets  

---

## 📈 Visualizations
- PnL vs Sentiment (Boxplot)  
- Win Rate by Sentiment (Bar Chart)  
- Trade Frequency by Sentiment
